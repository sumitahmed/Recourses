Database Encryption Techniques

1. Symmetric Key Encryption

Definition: In Symmetric Key Encryption, the same key is used for both encryption and decryption of data. This makes it fast and efficient for encrypting large datasets.

Algorithm Examples:

  • AES (Advanced Encryption Standard)
  • DES (Data Encryption Standard)

Use Case:

Encrypting stored backups or large files.

Example:

    Plaintext: SensitiveData123
    Encryption Key: Key123
    Encrypted Text: GZ%3@#F1^!

Process:

  1. Data is encrypted using a shared secret key.
  2. The same key is used to decrypt the data.

2. Asymmetric Key Encryption

Definition: In Asymmetric Key Encryption, two keys are used:

  • Public Key: Encrypts data.
  • Private Key: Decrypts data.

Algorithm Examples:

  • RSA (Rivest-Shamir-Adleman)
  • ECC (Elliptic Curve Cryptography)

Use Case:

Secure communication over networks (e.g., HTTPS, email encryption).

Example:

    Public Key encrypts: SensitiveData123 → H#@4G$%1^
    Private Key decrypts: H#@4G$%1^ → SensitiveData123

3. Transparent Data Encryption (TDE)

Definition: TDE encrypts the entire database at the file level, making it transparent to applications. Encryption and decryption occur automatically as data is written to and read from the database.

Use Case:

Protecting sensitive data like financial records.

Example:

Action Data
Plaintext Insert Sensitive123
Stored Encrypted Data 1G%3F#2!@

4. Column-Level Encryption

Definition: Encrypts specific columns in a database. This approach is highly granular, allowing sensitive data like credit card numbers to be encrypted while leaving other data unencrypted.

Use Case:

Encrypting PII (Personally Identifiable Information) like SSNs, credit card numbers, etc.

Example:

Name Credit Card (Encrypted)
Alice #E3G%5H1@
Bob H1#5F%6G2

5. File-Level Encryption

Definition: Encrypts individual files or directories. Commonly used to secure backups or exported database files.

Use Case:

Encrypting database export files to prevent unauthorized access.

Example:

    Plain File: backup.sql
    Encrypted File: backup_encrypted.sql

Tools:

  • GPG (GNU Privacy Guard)
  • AES file encryption utilities
